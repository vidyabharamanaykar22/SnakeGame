# 🐍 Classic Snake Game in Java Swing
<img width="593" height="621" alt="WhatsApp Image 2026-07-22 at 14 57 28" src="https://github.com/user-attachments/assets/e4a58d48-4f73-4c77-a14e-a37726890b41" />

A classic 2D Snake Game built using Java's `javax.swing` and `java.awt` GUI libraries. Navigate the snake using arrow keys, consume blue apples to increase your score and length, and avoid colliding with the boundaries or your own body!

---

## 🌟 Features

* **Retro 2D Graphics:** Built natively with Java Swing and AWT for smooth rendering.
* **Score Tracker:** Live updates of your current score during gameplay.
* **Responsive Controls:** Intuitive arrow key navigation with directional collision prevention (e.g., cannot move directly backwards into yourself).
* **Game Over Screen:** Displays final score upon collision with walls or self.

---

## 🎮 Game Controls

| Key | Action |
| :--- | :--- |
| **Up Arrow ($\uparrow$)** | Move Up |
| **Down Arrow ($\downarrow$)** | Move Down |
| **Left Arrow ($\leftarrow$)** | Move Left |
| **Right Arrow ($\rightarrow$)** | Move Right |
